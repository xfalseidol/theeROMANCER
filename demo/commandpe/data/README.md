# Data As Needed

## What's here

* ```commandpe_input``` - is a set of input files needed by ROMANCER to parse CPE files
* ```commandpe_output``` - is the output from a CommandPE scenario. Python
    code to artifically generate some is included.
* ```create_cpe_*.py``` - Python code to create sample CPE inputs/outputs
# ```targets.csv``` - a CSV input file used by create_cpe_*.py that describes targets
# ```weapons.csv``` - a CSV input file used by create_cpe_*.py that describes weapons
